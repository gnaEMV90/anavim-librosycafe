# Fotos de productos ANAVIM

Carpeta preparada para subir fotos reales de productos, promos, combos y extras de ANAVIM.

## Nombres recomendados

Usar nombres simples, en minúscula, sin espacios ni acentos:

```txt
combo-libro-cafe-01.png
biblia-infantil-01.png
cafe-grano-01.png
senaladores-01.png
promo-dia-del-padre-01.png
```

## Formatos recomendados

- `.jpg` o `.png` si la foto viene directa del celular.
- `.webp` si ya está optimizada.
- Evitar archivos muy pesados si no hace falta.

## Uso en la web

Las imágenes de esta carpeta se pueden referenciar así:

```txt
/images/productos/nombre-del-archivo.png
```

Ejemplo:

```js
imageSrc: '/images/productos/combo-libro-cafe-01.png'
```
